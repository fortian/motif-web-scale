What is this?

The Motif Mentor Smart Scale is a neat Bluetooth based scale that has poor app support. This web app does what everyone asks for when they get the scale:

"I just want to weigh stuff"

everyone

The scale is a Bluetooth Low Energy device, and Google Chrome has Web Bluetooth support, so I wrote this app to connect to the Motif Perfect scale and just do one thing well: weigh stuff.

Notes

  • The scale times out in 5 minutes, whether you are using it or not. Each time it does, the blue light on it will start flashing.
  • It is not possible to read the battery level AND weight at the same time. This is a limitation of BLE, not the scale. I'm working on a work-around.
